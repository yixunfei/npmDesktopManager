# npmDesktopManager

一个现代化、跨平台的 npm 包管理桌面应用，提供图形化界面管理项目依赖和全局包。

## 主要功能

- 🔍 **包搜索** - 快速搜索 npm 包，查看详细信息
- 📦 **依赖管理** - 管理项目和全局依赖
- 🌳 **依赖树分析** - 可视化依赖关系
- 🔐 **安全审计** - 检测并修复安全漏洞
- 🚀 **发布管理** - 发布包到 npm 或私有仓库
- 🎨 **主题切换** - 支持亮色/暗色主题

## 技术栈

Electron + React + TypeScript + Ant Design + Zustand

## 快速开始

```bash
# 安装依赖
npm install

# 生成图标（需要 ImageMagick）
npm run build:icons

# 启动开发服务器
npm run dev

# 构建应用
npm run dist
```

## 支持平台

- Windows 7+ (安装版 / 便携版)
- macOS 10.13+ (DMG / ZIP)
- Linux (AppImage / DEB / RPM)
